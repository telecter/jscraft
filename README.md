# jscraft

A simple, very basic recreation of Minecraft Classic written in three.js



### Controls

**W** Move forward  
**A** Move left  
**S** Move backwards  
**D** Move right  
**Space** Move up  
**Left Shift** Move down  

### Current "Features"

- Spectator-mode like movement; no physics
- Block breaking and placing
- Proper world generation (soon!)
